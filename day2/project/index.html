HTML & CSS Basics Summary

HTML Summary

HTML (HyperText Markup Language) is used to define our page content, structure and meaning. You dont use it for styleing purpose.Use CSS for that instead !

  • HTML Uses "elements " to describe content
  • You can also have empty elements like images
  • You can also configure elements with attributes

    • Learn more about all available HTML elements on the MDN HTML property referecens

    CSS Summary

    CSS (Cascading Style Sheets) is used for styling you page content Style are assigned via property value pairs You can assing styles via the "style" attribute.

  • To avoid code duplication use gloval styles
  • When working with CSS , concepts like inheritane specificity and the box model should be understood
